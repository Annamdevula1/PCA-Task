# PCA-Task
Principal Component Analysis (PCA) on Wine Dataset

Objective

The objective of this project is to apply Principal Component Analysis (PCA) to the Wine Dataset for dimensionality reduction and data visualization. PCA helps reduce the number of features while preserving the most important information in the dataset.

Tools and Libraries

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

Dataset

Wine Dataset (CSV format)

Project Steps

1. Load the Wine Dataset.
2. Separate features and target column.
3. Standardize the data using StandardScaler.
4. Apply PCA with 2 principal components.
5. Analyze the explained variance ratio.
6. Visualize the transformed data using plots.
7. Save the transformed dataset.

Results

- Reduced 13 original features to 2 principal components (PC1 and PC2).
- Visualized the dataset in a 2D space.
- Analyzed the variance explained by each principal component.
- Created a transformed dataset for further analysis.

Deliverables

- Transformed Dataset (transformed_wine_dataset.csv)
- Explained Variance Plot
- PCA Scatter Plot
- Jupyter Notebook
- README.md
